Considerations About the Program

I made this dedicated web browser for listen music on Deezer named Playzer. For this program, under same folder, there's a text file for replace my profile page link by yours and a colors text file for customize audio visualization I added. To note, there's handle to key F11 to set Playzer in fullscreen. It comes with another program, when having a laptop PC and Bluetooth speakers, to enable both. There's a parameters file under same folder of the program if there is a error message, it need to set the parameter to True or False, it's made if first device is the capture device. To note, it ask to set a audio quality of 16 bits and 44100 Hz format for both audio cards, and enable both material acceleration and enhanced audio, under audio settings, but there's also another parameter to disable the program to not automatically launching. It also need to set level of earing for laptop speakers to around 10 for avoid too loud sound. My source codes and program are here.
